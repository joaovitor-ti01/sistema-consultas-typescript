#   Sistema de consultas Typescript

## Sobre o Projeto

Fizemos o projeto como o objetivo de integrar a modelagem TypeScript do sistema ao aplicativo mobile, organizando melhor os tipos e utilizar essa modelagem dentro do React Native

## Como Executar o Projeto 

1. Clone este repositório:

   ```

    git clone https://github.com/mig2505/sistema-consultas-typescript.git

   ```

2. Abra o projeto em sua IDE de preferência
